# 🌐 Cloudflare Worker VLESS 中转服务说明文档

## 📌 项目简介

本项目是一个基于 **Cloudflare Workers** 的轻量级 VLESS 中转服务，支持以下特性：

* ✅ 支持 VLESS over WebSocket 中转
* ✅ 支持 UUID 验证
* ✅ 多 IP 随机分流
* ✅ 自动生成订阅链接
* ✅ 支持 NAT64 IPv6 网络
* ✅ 支持备用反代 IP 回退

适用于绕过封锁、优化连接质量、实现国内外网络中转等多种场景。

## 🛠️ 部署指南

### 1️⃣ 创建 Worker

1. 登录 Cloudflare → 进入 **Workers & Pages**
2. 点击 **Create Application** → 选择 **Worker**
3. 在编辑器中粘贴脚本（使用优化版代码）
4. 点击 **Save and Deploy** 完成部署

### 2️⃣ 设置环境变量（推荐）

路径：Worker → `Settings` → `Variables` → 添加以下变量：

| 变量名       | 类型  | 说明                           |
| --------- | --- | ---------------------------- |
| `UUID`    | 字符串 | 用于连接验证（**必填**）               |
| `ID`      | 字符串 | 自定义订阅路径标识，如 `123456`（**必填**） |
| `IP`      | 字符串 | 出口 IP 列表，多个 IP 使用换行分隔        |
| `TXT`     | 字符串 | 可选，订阅附加信息（备注/公告等）            |
| `PROXYIP` | 字符串 | 备用反代地址，例如 `1.2.3.4:443`      |
| `启用反代功能`  | 布尔值 | 是否启用备用反代（填写 `true` 启用）       |
| `NAT64`   | 布尔值 | 是否启用 NAT64（填写 `true` 启用）     |
| `我的节点名字`  | 字符串 | 节点名称（展示在订阅中，默认值为“狂暴”）        |

📌 示例配置：

```ini
UUID = 5aba5b77-48eb-4ae2-b60d-5bfee7ac169e
ID = 123456
IP = 1.1.1.1
PROXYIP = sjc.o00o.ooo:443
启用反代功能 = true
NAT64 = true
我的节点名字 = 狂暴中转
```

## ❓ 常见问题解答（FAQ）

### 1. 客户端连接失败，可能的原因？

* ✅ 检查 UUID 是否与环境变量配置一致
* ✅ IP 被封：尝试更换 `IP` 变量中的地址
* ✅ 网络为 IPv6-only：尝试关闭 `NAT64`
* ✅ 备用反代失效：确认 `PROXYIP` 可用性

### 2. 支持哪些客户端？

* ✅ v2rayN（Windows）
* ✅ v2rayNG / SagerNet（Android）
* ✅ Shadowrocket / Stash（iOS）
* ✅ Clash.Meta / MetaX（全平台支持）
